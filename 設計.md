# 設計.md — VISION BASE Quartz 視覺規範

這個網站（[`vision-base-quartz`](https://github.com/visionbase-usc/vision-base-quartz)）的視覺套用 **VISION BASE 品牌識別系統**，與 VISION BASE 簡報公版（`vision-base-presentation` skill）同一套標準，來源為品牌規範書 `20220114_VISION_BASE_guideline.pdf`。

本文件說明把品牌規範對應到 Quartz 的決策，方便日後維護。

---

## 1. 核心原則

| 原則 | 說明 |
|---|---|
| **深色為主** | 預設 Base Black `#0F0F0F` 背景、Vision White `#FFFFFF` 文字。網站預設即為夜間版本。 |
| **克制用色** | 約 70% 黑 / 25% 白 / **5% accent**。accent 只用在連結、active 狀態、引用線。 |
| **建築感排版** | 大標題、緊字距、充足留白、清楚層級。 |
| **品牌標誌存在感** | 左側欄頂端放 VISION BASE wordmark（取代純文字站名）。 |

---

## 2. 色彩對應（品牌 token → Quartz 色彩角色）

Quartz 用 8 個色彩角色（`light` / `lightgray` / `gray` / `darkgray` / `dark` / `secondary` / `tertiary` / `highlight` / `textHighlight`）。對應如下，設定於 [`quartz.config.yaml`](quartz.config.yaml) 的 `theme.colors`。

### 深色（預設，Base Black 變體）

| Quartz 角色 | 值 | 品牌 token / 用途 |
|---|---|---|
| `light`（頁面底色） | `#0F0F0F` | Base Black |
| `dark`（標題文字） | `#FFFFFF` | Vision White |
| `darkgray`（內文） | `#E8E8E8` | 近 Vision White，長文較舒適 |
| `gray`（次要 / 圖譜線） | `#8A8A8A` | Muted Text |
| `lightgray`（分隔線） | `#2A2A2A` | Divider |
| `secondary`（連結 / accent） | `#F96419` | **VB Orange** — 主 accent |
| `tertiary`（hover） | `#3DA1D3` | **VB Blue** |
| `highlight` | `rgba(249,100,25,0.12)` | 橘色調的內部連結 / 程式碼底 |
| `textHighlight`（`==標記==`） | `#F9641955` | VB Orange 半透明 |

### 淺色（反相 Vision White 變體）

| Quartz 角色 | 值 | 說明 |
|---|---|---|
| `light` | `#FFFFFF` | Vision White 底 |
| `dark` | `#0F0F0F` | Base Black 標題 |
| `darkgray` | `#2B2B2B` | 內文 |
| `secondary` | `#D2520F` | VB Orange 加深，確保白底對比 |
| `tertiary` | `#2B86B4` | VB Blue 加深 |
| `textHighlight` | `#E6E14366` | VB Yellow 標記 |

> 完整 accent 盤（備用，勿濫用）：Orange `#F96419`、Blue `#3DA1D3`、Red `#F21847`、Yellow `#E6E143`、Purple `#5F4ADD`、Green `#6AD868`。

## 3. 字體

品牌規範：英文 **Avenir Next**、中文 **Noto Sans TC（思源黑體）**。

問題：**Avenir Next 不是免費 webfont**，無法從 Google Fonts 載入。處理方式：

- 字體堆疊（family stack）把 `"Avenir Next"` 排第一 → **Mac / iOS 訪客（實驗室多數）會看到真正的品牌字**。
- 其後接 **Manrope**（Google Fonts，幾何無襯線，最接近 Avenir 的免費替代）作為其他平台的 web 後備。
- 再接 **Noto Sans TC** → 所有中文字（CJK）都落到思源黑體。

實際堆疊（定義於 [`quartz/styles/custom.scss`](quartz/styles/custom.scss)）：

```
"Avenir Next", "Manrope", "Noto Sans TC", -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif
```

- Manrope / Noto Sans TC / IBM Plex Mono 的實際載入由 `fonts` 外掛負責（見 `quartz.config.yaml` 的 fonts plugin `options`）。
- `custom.scss` 覆寫 `--headerFont` / `--bodyFont` 變數，確保堆疊順序正確。
- 程式碼字體：**IBM Plex Mono**。
- Logo 字體 Mattone 為品牌保留字，僅用於 logo 圖檔，不用於內文。

## 4. 標誌（Logo）

| 檔案 | 用途 |
|---|---|
| [`quartz/static/vb/vb-logo-on-black.png`](quartz/static/vb) | 白字 wordmark（黑底）。深色模式用，靠 `mix-blend-mode: screen` 讓黑底融掉、只留白字。 |
| [`quartz/static/vb/vb-logo-black.png`](quartz/static/vb) | 黑字 wordmark（透明底）。淺色模式用。 |

放置：左側欄頂端 `.page-title` 區，以背景圖呈現、文字以 `text-indent` 移出畫面（保留給 a11y / SEO）。底部有一條 `lightgray` hairline 分隔。

> 為何用 `mix-blend-mode: screen`：品牌提供的白字 logo 是「白字＋純黑底」PNG，沒有透明背景版。screen 混合模式會讓純黑像素透出底色、白字保留，等於即時去背，不需另外做透明圖。

## 5. 其他品牌套用

- **預設夜間模式**：由 [`visionbase-usc/quartz-darkmode-vb`](https://github.com/visionbase-usc/quartz-darkmode-vb) 外掛達成（無偏好時預設 dark）。
- **引用區塊**：左側 3px VB Orange 線。
- **連結**：Orange，hover 轉 Blue。
- **頁尾**：一行品牌字「VISION BASE Lab · College of Design, Shih Chien University 實踐大學設計學院」+ org / vault 連結。
- **社群卡片（og-image）**：自動沿用上述色彩與字體。
