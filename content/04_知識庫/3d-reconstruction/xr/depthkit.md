# Depthkit

[Depthkit Core](https://www.depthkit.tv/depthkit-core) 是用於 volumetric video（體積影像）擷取、處理與部署的工具集，可整合 Azure Kinect、Femto Bolt 等深度感測器，輸出可用於 Unity / Unreal / Web 的體積影像資產。

## 用途

- 體積影像（volumetric capture）製作流程
- XR / VR / AR 內容中的真人體積化呈現
- 多視角擷取與重建

## 相關資源

- 官方網站：<https://www.depthkit.tv/depthkit-core>
- Depthkit Studio：多相機同步擷取
- Depthkit Cinema：高品質離線重建
