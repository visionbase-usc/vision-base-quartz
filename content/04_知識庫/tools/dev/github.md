# GitHub

開發版本控制平台，用於 code repo、issue 追蹤、PR review、CI/CD 工作流。

## 常用入口

- 個人 / 組織 repos 清單：`https://github.com/<org>?tab=repositories`
- 各 repo 的 Settings → Actions 設定 CI；Settings → Secrets 管理 token

- 待補：CI/CD workflow 範例、PR template、commit convention
